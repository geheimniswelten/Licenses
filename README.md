# Software Licenses

Eine kleine Zusammenstellung bekannter Lizenzen. \
Es ist nicht auf Vollständigkeit ausgelegt (gibt eh zu Viele), \
ebenso nicht ausschließlich auf OpenSource. (manchmal möchte man auch Software ohne Code veröffentlichen)

Aktuell sind vor allem die Lizenzen gelistet, welche GitHub zur Auswahl anbietet, zusätzlich deren weiteren Versionen. \
Dann noch einige Multilizenzen und was sonst noch durch Abhängigkeiten hinzu kam.

Die einelnen Lizenzen stehen jeweils unter ihrer eigenen Lizenz, bzw. Einer der jeweiligen Ersteller.

Kompatibilitäten, Unterschiede, Gemeinsamkeiten, Abhängigkeiten usw. werden versucht noch zu füllen. \
Es gibt ein Ranking durch GitHub. Einmal Anhand der Auswahl, welche der Editor beim Erstellen des Repositories oder dem Hinzufügen einer LICENSE-Datei gegeben ist, sowie über die Anzahl deren Verbreitung.

Überall sind einheitlich die `{:Platzhalter:}` markiert. \
Einige lassen sich als Markdown `.md` und Text `.txt` einbinden \
und wo es scheiße aussieht, das gibt es nur als `.txt`.
